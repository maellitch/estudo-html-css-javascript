Créditos de Mídia

Este projeto foi desenvolvido para fins de estudo e utiliza imagens/vídeos para ilustrar as funcionalidades.
Texto presente gerado por IA podendo conter erros, notícias falsas entre outros.

* **Imagens:** Obtidas via Pexels.
* **Tecnologias de Mídia:** Uso de elemento `<picture>` para responsividade avançada.
